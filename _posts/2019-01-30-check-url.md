title: [Android] Webview 올바른 Url 인지 체크하기
categories: Android
---

- 프로토콜이 없으면 붙이기
``` java
URLUtil.guessUrl("url")
```

- 올바른 url 형식인지 체크
``` java
URLUtil.isValidUrl("url")
```

- 프로토콜을 붙이지 않은 상태에서 url 인지 체크 (검색엔진에 사용할지 분기처리할 때 사용)
``` java
Patterns.WEB_URL.matcher(url).matches()
```
